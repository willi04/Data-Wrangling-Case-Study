# Data Wrangling – Case Study (R)

Projet complet de nettoyage et transformation de données réalisé en langage R.

## Objectif
Traiter un jeu de données brutes (data_row), identifier et corriger les incohérences,
les valeurs manquantes et les doublons, puis produire un jeu de données propre (data_clean)
prêt pour l'analyse.

## Structure du projet
- `data_row/` – données brutes d'entrée
- `data_clean/` – données nettoyées en sortie
- `scripts/` – scripts R de traitement et transformation

## Technologies
- Langage : R
- Environnement : RStudio
